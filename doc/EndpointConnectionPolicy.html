POST /api/v1/policies/type/EndpointConnection

Create a new Endpoint Connection between different DTNs.

Request JSON Object
  • deadline (string) - Time user wants to finish a transfer. String should be in RFC3339 format: "2017-04-12T23:20:50"
  • srcendpoint (string) - One endpoint on this connection. Must point to a DTN node in the network topology.
  • dstendpoint (string) - Other endpoint on this connection. Must point to a DTN node in the network topology.
  • dataquantity (int) - Amount of data to be transferred in bytes.
Response JSON Object
  • policy (dict) - Link to the newly created policy
Status Codes
  • 201 Created - Policy successfully created
  • 400 Bad Request - Invalid Argument
  • 401 Unauthorized - Returned if a user is not able to use a particular policy type or particular resource that this policy requires
  • 404 Not Found - Returned if a resource does not exist that is being requested.
Example Request

POST /api/v1/policies/type/EndpointConnection

{

  "EndpointConnection":{

    "deadline":"1985-04-12T23:20:50",

    "srcendpoint":"atlh1",

    "dstendpoint":"miah2",

    "dataquantity":57000000000

  }

}
